# How to run 

## 1. Clone Repository
```
git clone <link>
cd marginal_value
```

## 2. Install Dependencies
```
pip install tensorflow datasets numpy pandas matplotlib
```

## 3. Download Dataset of 1000
```
python marginalvalue.py
```

## 4. Test Training
```
python train.py --n 500 --seed 0
```

## 5. Run 
```
python run.py
```

## 6. Graph
```
python graph.py
```
