This repository contains ongoing research in Spring 2026 exploring **data as economic infrastructure in AI systems**. The work investigates how the quantity and distribution of data influence model behavior, performance, access, and downstream outcomes.

The project examines the economic role of data in modern machine learning pipelines, focusing on questions such as:

- How does increasing training data affect model performance?
- What is the marginal value of additional data?
- How does data availability shape access to AI capabilities?
- What implications does data concentration have for equity and system outcomes?
